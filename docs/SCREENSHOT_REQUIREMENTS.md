# Screenshot Requirements

For every UI task, provide screenshots for:

1. Desktop width: 1440px
2. Narrow width: 390px, only if responsive layout changed

Required screenshots:

- Page initial state
- Main interaction state
- Empty state
- Error state, if relevant

For sale flow tasks, include screenshots of:

- Empty sale
- Search results
- Cart with multiple products
- Confirmed sale state

Use Playwright whenever possible.
